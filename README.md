## Доступные команды
| Команда | Описание |
|---------|----------|
| `npm install` | Установливает зависимости |
| `npm run start` | Запускает проект и открывает веб-сервер, и следит за изменениями |
| `npm run build` | Собирает билды по рекламным сеткам |

##  Используемые версии

* pixi 7.2.0
* tweedle.js 2.0.1
